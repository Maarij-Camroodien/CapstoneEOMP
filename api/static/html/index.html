Home

HTTP Methods Paths Descriptions
GET / or /home This router takes you to the Home Page of the API

Users

HTTP Methods Paths Descriptions
GET /users This router shows all the users in the database
GET /user/:id This router shows a single user when a user requests a ID
POST /user This router allows us to register a new user into the database
POST /login This router allows us to login to the website
PATCH /user/:id This router allows us to update a users details in the database
DELETE /user/:id This router allows us delete user and their details in the database

Products

HTTP Methods Paths Descriptions
GET /products This router shows all the products in the database
GET /product/:id This router shows a single product when a user requests the product by ID
POST /product This router allows us to add a new product into the database
PATCH /product/:id This router is used to update a products and its details on the database
DELETE /product/:id This router is used to delete a products from the database
